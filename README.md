## Tic Tac Toe Game

A modern and interactive Tic Tac Toe game built with React.
This project allows two players to play the classic game with dynamic game state updates and winner detection.

## Features
Two-player gameplay
Dynamic turn switching
Winner detection
Draw detection
Game reset functionality
Responsive UI
Clean component-based architecture

## Tech Stack
React.js
JavaScript (ES6)
CSS3
Vite
